全局loading

应用场景1：接口未返回数据，需要展示加载状态
应用场景2：页面出现问题，需要过渡
应用场景3：其他页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。
注：计算组件，可以在连线后的逻辑组件中找到

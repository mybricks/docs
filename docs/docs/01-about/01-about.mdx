# 关于 MyBricks

MyBricks 为用户提供了两种灵活的使用方式：应用开发平台和独立引擎。

应用开发平台内置了「中后台系统」、「小程序」、「H5页面」、「主题包」、「云组件」等应用，也支持扩展更多自定义应用。

另外也可以基于 MyBricks Inside 计划，快速定制自己的低代码平台。

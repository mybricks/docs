# 关于 MyBricks

MyBricks 为用户提供了两种灵活的使用方式：aPaas 平台和独立引擎。

aPaas 平台内置了「中后台系统」、「小程序」、「H5 页面」、「主题包」、「云组件」等应用，也支持自定义扩展更多应用。

另外也可以基于 MyBricks 引擎，快速定制属于自己的低代码平台。

---
title: 服务管理
---

本篇主要介绍如何对「平台服务」进行管理，启动 / 停止 / 重启服务

:::tip
平台服务本质上是使用 [PM2](https://pm2.keymetrics.io/) 进行管理，所以也可以直接使用 PM2 的各种命令
:::

## 直接使用命令管理

进入到项目根目录后，可以使用各类 npm run xxx 命令

### 启动 / 重启服务
```bash
npm run start
```

### 终止服务
```bash
npm run stop
```

## 使用 PM2 辅助管理
各类 pm2 的命令都可以使用

:::tip
服务的启停还是建议使用上方的命令，因为其中还包含一些检测脚本
:::

### 查看进程状态
```bash
pm2 ls
```

## 使用问题

### 在搭建中，使用保存功能出错
1. 错误信息提示 Got a packet bigger than'max allowed_packet' bytes
2. 错误信息提示 write EPIPE

可以查看是否是 MySql的配置问题，主要看 max_allowed_packet 配置
> 可参考此链接 https://stackoverflow.com/questions/8062496/how-to-change-max-allowed-packet-size

## 部署问题

### 安装应用后，能打开页面，但是显示一直加载中
请先检查「物料中心」应用是否安装，没有的话先安装此应用

### 安装应用后服务未成功重启，导致模块加载异常，同时 PM2停止服务时，网站依然存在

使用以下命令，查看当前运行的 PM2 进程有多少个，如果有多个，则为异常情况，删除掉多余的进程即可
```bash
ps aux | grep -e 'pm2'
```

## 新建方案

常规埋点系统包含「**全局级别的埋点**」和「**事件级别的埋点**」

- 全局级别的埋点：往往SDK引入后会自动上报、比如 PV、UV、FMP 等页面级埋点
- 事件级别的埋点：元素的点击、曝光以及各类自定义事件等

在MyBricks中，一个埋点方案包含了上面的所有功能，我们可以通过新建来创建一个埋点方案

## 事件埋点配置
在搭建系统中，事件级别的埋点往往需要依托于组件存在，我们可以通过定义每个组件的上报逻辑来完成埋点

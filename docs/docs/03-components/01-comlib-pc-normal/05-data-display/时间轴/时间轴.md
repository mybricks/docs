> 垂直展示的时间流信息\
> 应用场景1：当有一系列信息需按时间排列时，可正序和倒序。\
> 应用场景2：需要有一条时间轴进行视觉上的串联时。

Demo地址：[【时间轴】基本使用](https://my.mybricks.world/mybricks-app-pcspa/index.html?id=514748888363077)

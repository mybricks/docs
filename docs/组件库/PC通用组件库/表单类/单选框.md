# **逻辑编排**
### **值初始化事件**
![](radio-imgs/img9.png)

输出时机：

1. 给单选框的“**设置初始值**”输入项设置数据
2. 给单选框所在表单容器的“**设置表单数据**”输入项设置数据

输出内容：
单选框的当前值

### **值更新事件**
![](radio-imgs/img10.png)

输出时机：

1. 给单选框的“**设置值**”输入项设置数据
2. 给单选框所在表单容器的“**设置表单数据(触发值变化)**”输入项设置数据
3. 用户操作单选框的增删改操作时

输出内容：
单选框的当前值
说明：
可以用于实现监听联动
